# 🤖 AI Chat UI (JavaScript)

A modern AI chat interface built with HTML, CSS, and JavaScript.
This project simulates a ChatGPT-like experience with typing animation, smart replies, and persistent chat history.

---

## ✨ Features

* Chat interface (user + AI messages)
* Typing animation for bot replies
* Smart predefined responses
* Chat history using localStorage
* Auto-scroll behavior
* Clean modern UI

---

## 🛠 Technologies Used

* HTML5
* CSS3 (Flexbox)
* JavaScript (DOM + localStorage)

---
